## Konfigurationssettings

Für Meshcore-Konfigurationen empfehlen wir die folgenden Einstellungen:

- `set flood.max.unscoped 5`
- `set flood.max 16`
- `set flood.max.advert 16`
- `set path.hash.mode 1`
- `set dutycycle 10`
- `set loop.detect moderate`
- `set txdelay 0.5`
- `set direct.txdelay 0.2`
- `set rxdelay 0.0`
- `set flood.advert.interval 25`
- `set advert.interval 60`
